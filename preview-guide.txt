TOOLBOX PAY - PREVIEW & EDITING SPECS
-------------------------------------
PRIMARY COLOUR: Orange (#FFA500)
SECONDARY COLOUR: White (#FFFFFF)
ACCENT: Charcoal (#151515)

PREVIEW STEPS:
1. Open index.html in a modern browser (Chrome/Edge preferred).
2. Use 'F12' or 'Inspect' to toggle Mobile View.
3. Set orientation to 'Portrait' to mimic the trade-use environment.
4. Ensure 'auth.js' is in the /src/ folder or security redirects will trigger.

EDITING WORKFLOW:
1. Review the UI on the mobile emulator.
2. Note any alignment issues or text changes.
3. Request the "Complete Block" for the specific page from Gemini.
4. Replace the old code entirely with the new block.
